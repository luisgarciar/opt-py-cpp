:mod:`simpleopt.opt` - Simple optimization methods
===========================================================================
The :mod:`simpleopt.opt` module provides implementations of the steepest descent and
conjugate gradient methods for solving unconstrained optimization problems,
as well as a simple class for defining and storing optimization problems.
